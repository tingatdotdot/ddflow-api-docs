ddflow API Documentation

薪資墊付系統 API 文件

ddflow 發送的 Webhook (Outbound)

AIBOU Webhook API

定義 AIBOU 系統需實作的 Webhook 端點，用於接收 ddflow 系統發送的撥款完成與逾時通知。 包含撥款完成通知 (Disbursement Complete) 和逾時通知 (Timeout) 兩種類型。

Outbound Webhook 版本: 1.0.0 2 個端點

ddflow 接收的 Webhook (Inbound)

ALPHA Webhook API

定義 ddflow 系統提供給 ALPHA 呼叫的 Webhook 端點。 當 LINE 確認完成或逾時時，ALPHA 系統會呼叫這些端點通知 ddflow。

Inbound Webhook 版本: 1.0.0 3 個端點

Bank Webhook API

定義銀行/ACH 服務呼叫 ddflow 的 Webhook 端點。 包含 VAC 付款通知和 ACH 轉帳結果通知兩種類型。

Inbound Webhook 版本: 1.0.0 2 個端點

VAC Webhook API

定義 VAC 服務 (tpg-rc.qrpay.tw) 呼叫 ddflow 的付款通知端點。 當虛擬帳號收到付款時，VAC 服務發送付款狀態通知。

Inbound Webhook 版本: 1.0.0 1 個端點

Partner API (第三方合作夥伴)

Partner API

提供給第三方合作夥伴查詢薪資墊付相關資訊的 API。 支援查詢額度、訂單列表、訂單狀態及處理歷程。使用 OAuth2 Client Credentials 認證。

Partner 版本: 1.0.0 4 個端點
